  1. Otwórz terminal (np. Win + R → cmd)
  2. Przejdź do folderu projektu: 
    cd C:\Users\TwojaNazwaUzytkownika\source\repos\Solitaire\x64\Konfiguracja
  3. Uruchom plik wykonywalny: 
    Solitaire.exe
Gra powinna się uruchomić w oknie terminala 🎮.

🎮 Jak grać w grę

Poniżej znajdziesz listę dostępnych komend w grze wraz z ich opisem:

  • wyjdz - wychodzi z gry
  • reset - resetuje grę
  • przetasuj - przetasowuje karty, gdy wszystkie karty zostały dobrane
  • dobierz lub d - dobiera kartę z puli
  • przenies lub p [od_nr_kolumny] [do_nr_kolumny] [ilość_kart] - przenosi wskazaną liczbę kart z jednej kolumny do drugiej
  • z_puli_do_kolumny lub pk [nr_kolumny] - przenosi dobraną kartę z puli do kolumny
  • z_puli_do_rezerwy lub pr [nr_rezerwy] - przenosi dobraną kartę z puli do rezerwy
  • z_kolumny_do_rezerwy lub kr [nr_kolumny] [nr_rezerwy] - przenosi kartę z kolumny do rezerwy
  • z_rezerwy_do_kolumny lub rk [nr_rezerwy] [nr_kolumny] - przenosi kartę z rezerwy do kolumny
  • menu uruchamia główne menu
  • pomoc wyświetla liste komend
  • zapisz [nazwa zapisu] - zapisuje grę

Przykład użycia: p 3 5 2 - przenosi 2 karty z kolumny 3 do kolumny 5.
