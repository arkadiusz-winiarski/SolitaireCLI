◆ isValidFilename()

bool BufferedIO::isValidFilename ( const std::string & filename)
inline

Checks if a given filename string is valid for saving a file.

This function validates the filename according to common restrictions:

  • Must not be empty.
  • Must not contain illegal characters (such as \ / : * ? " < > |).
  • Must not consist only of spaces or dots.
  • Must not exceed typical filename length limits (255 characters).
Parameters
filenameThe filename string to validate.
Returns
true If the filename is valid and good to use for saving a file.
false If the filename is invalid.
Note
This function performs basic validation primarily suitable for Windows filenames. Additional OS-specific checks (like reserved names) are not included.
