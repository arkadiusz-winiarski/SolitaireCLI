Detailed Description

Builds and manages multi-line wide strings supporting character-level ANSI color codes.

This class stores multiple lines of wide characters (std::wstring) along with separate color layers for each character, allowing fine-grained control of color output. It supports parsing of embedded ANSI escape sequences and renders the combined colored string with minimal reset codes.
